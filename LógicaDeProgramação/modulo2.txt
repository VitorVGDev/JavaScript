Módulo 2: Lógica de Programação com JavaScript

→ Estruturas de Decisão: if, else, else if

let idade = 18;

if (idade < 18) {
    console.log("Você é menor de idade");
} else if (idade >= 18 && idade <= 60) {
    console.log("Você é adulto");
} else {
    console.log("Você é idoso.");
}
